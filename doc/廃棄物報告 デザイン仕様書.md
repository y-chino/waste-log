# **廃棄物報告 デザイン統合仕様書**

作成日: 2026/03/22

バージョン: v1.0.8

## **1\. 概要**

本システムは、現場作業中の「片手操作」と「瞬時の視認性」を最優先とした、高密度かつ高レスポンスなモバイル・ファースト・デザインを追求する。

## **2\. デザインコンセプト**

* **One-Handed** : 主要アクションを下部に集約し、親指の可動範囲で完結。  
* **High Contrast** : 現場照明下でも読み取れる太字（Black 900）と配色。  
* **Visual Feedback** : 履歴の視認性、ローディング体験、PC向けナビ操作性を継続的に改善。

### **3.1 特殊テクスチャ (today-striped)**

* **スタイル** : 135度の角度で薄緑と濃い緑の細い斜線を繰り返すグラデーション。  
* **適用対象** : 「今日」の日付セル、および「ロック解除状態」のバッジ背景。

## **4\. タイポグラフィ**

フォントは **Noto Sans JP** を使用。

* **入力用巨大数値** : 32px (Black 900\)  
* **アプリタイトル** : 24px (Black 900\)  
* **カード内重量数値** : 30px (Black 900\)  
* **日時フォーマット** : 「月/日 時:分」（例: 03/22 09:15）

## **5\. コンポーネントおよびインタラクション仕様**

### **5.1 カレンダー (Ultra High-Density)**

* **実績の可視化** :  
  * 報告済み：bg-emerald-50  
  * 選択中：border-2 border-emerald-600  
  * 本日：today-striped  
* **遅延ロード通知** : 3ヶ月以上前のデータ取得時、中央にスピナー付きの「読み込み中...」オーバーレイを表示。  
* **ジェスチャー** : 水平スワイプにより月の切り替えアニメーション。

### **5.2 廃棄物カードと履歴表示**

* **場所インジケーター** : 所属内の場所数に応じたドットを表示。現在位置を強調。  
* **水平スワイプ** : カードエリアをスワイプし、同一所属内の「場所」を高速切り替え。
* **PC向け場所切り替え** : 場所ナビバー左右の `‹` / `›` ボタン、およびキーボード `←` `→` キーで場所を切り替え可能。複数場所がある場合のみボタンを表示し、端では非活性（opacity 30%）になる。カレンダーの月移動ボタンとボタン位置を縦に揃えて視覚的統一感を確保。  
* **詳細ヒント** : 右上の fa-info-circle タップで bg-slate-800 の吹き出し表示。  
* **登録履歴表示 (History View)** :  
  * アイコン：報告完了の緑枠チェックを日時行と水平（top-\[-2px\]）に配置。  
  * 日時：英語表記を排除し、日本語形式に統一。

### **5.3 計量パネル (Slider Panel)**

* **アクティブ強調** : 操作対象フィールドの背景を bg-blue-100 に、下線を実線に変更。  
* **垂直密度** : ボタン、スライダー、確定ボタンの余白を py-1.5 で統一。

### **5.4 説明ページ (Description View)**

* **余白調整** : Step 1（日付選択）上部の余白を拡大しヘッダーと分離。Step 3（まとめて報告）下部を詰め、アクションへの誘導を強化。

### **5.5 トースト通知 (Toast Hierarchy)**

* **下部 (Bottom)** : 成功・情報。指の届きやすい位置。  
* **上部 (Top)** : 警告・エラー。視認性を最大化。
